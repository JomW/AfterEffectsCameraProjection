用于DCC软件中的参考FBX文件，用于导入DCC软件辅助动画制作，可以进行整体位移，避免缩放和调整相对位置，避免调整摄像机参数

目前参数导出程序仅适配了MAX，其他的DCC软件还在适配中。
导入后请将输出分辨率设置为2000*1600，或是同比例下的其他分辨率/


Reference FBX file for DCC software, used to import into DCC software to assist with animation production. It allows for overall displacement while avoiding scaling and adjusting relative positions, as well as preventing adjustments to camera parameters.

Currently, the parameter export program is only compatible with MAX; other DCC software are still being adapted.

After importing, please set the output resolution to 2000*1600 or another resolution of the same aspect ratio.
